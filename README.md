# workstation

From your home directory execute one of the following:

## Initial clone
```
git init
git remote add origin https://github.com/hsiliev/workstation-scripts.git
git fetch
git checkout -t origin/master
```

## Subsequent clones
```
git init
git remote add origin https://github.com/hsiliev/workstation-scripts.git
git fetch
git reset origin/master
git checkout -t origin/master
```
